# 🛡️ Omni-Common: 通用安全与准则 (Base Layer)

## 🛡️ 核心合规
- **敏感词库**：禁止出现“最、第一、秒杀、赚钱、死、暴利”等。
- **合规标注**：AI 生成的插图必须主动标注“AI 辅助生成”。

## ⚖️ 事实对齐 (Fact Consistency)
- **角色一致性**：慕夕歌必须倔强，顾栖月必须清冷，苏梦城严禁穿甲。
- **背景一致性**：天门关战局、药路暗线等核心剧情点在分发时严禁冲突。

## 📂 资产规范
- **路径重映射**：自动转换物理路径至通用变量（如 `~/`）。
- **缓存策略**：临时资产归档至 `tmp/`，确认发布后迁移至 `assets/`。
